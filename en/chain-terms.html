Blockchain

memo

It is a string provided by JADEPOOL to user for recognizing deposits. For EOS and CYB, obtaining the deposit address is to get the hot wallet account of JADEPOOL and a deposit memo. Whether user transfers tokens from his/her own account or from other exchanges to JADEPOOL, the deposit memo provided by JADEPOOL should be correctly filled in the transaction.

utxo

dust

Dust is the lowest amount that can be include in a transaction. That is, any input and output value of a transaction must be higher than this value. Otherwise the transaction may fail to be broadcasted on blockchain. Dust is an unmodifiable configuration in JADEPOOL. Among the crypto assets supported by JADEPOOL, only BTC, LTC, and QTUM have dust configuration.
